## validator-generator

Canonical skill definition: `.claude/skills/validator-generator/SKILL.md`

This skill produces typed, assert-based validator functions from schema inputs so response checks stay centralized and reusable. It helps enforce consistent validation style across generated and hand-written tests. The output is intended to align with the framework's validator conventions in `src/validators.py`.

The folder-based `SKILL.md` is the source of truth for behavior, inputs, options, and constraints.
