## testcase-generator

Canonical skill definition: `.claude/skills/testcase-generator/SKILL.md`

This skill generates reviewer-ready API test-case spec sheets from multiple source types such as URL, OpenAPI, Jira, Confluence, and local documents. It focuses on normalized endpoint extraction and structured test-case output suitable for downstream automation steps. It also supports incremental updates via comparison workflows.

The folder-based `SKILL.md` is the source of truth for behavior, inputs, options, and constraints.
